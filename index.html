Civet

Painless data integration for your React application

Easy to Use

Civet takes care of the data handling. All you have to do is to bring it to the screen.

Flexible

Civet leaves the connection of the data source to you, allowing you to integrate nearly any backend.

Powered by React

Civet's functionality is accessible by React components and hooks, which makes it super easy to integrate it in your environment.

Learn How

The resource component is used to access your data, which you can then hand over to your display components.
